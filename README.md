# Проект Hakoton

Сайт для отслеживания мероприятий и личного рейтинга студентов.

## Как запустить на своем компе:

1. **Библиотеки**
   Сначала поставь все нужные пакеты:
   ```
   pip install -r requirements.txt
   ```

2. **База данных**
   Если при запуске ругается на отсутствие таблиц или колонок (OperationalError), нужно обновить базу командами:
   ```
   python manage.py makemigrations
   python manage.py migrate
   ```

3. **Запуск**
   Просто стартуй сервер.
   * На Windows: `python manage.py runserver`
   * На Mac: `python3 manage.py runserver`

## Куда заходить:
* Главная с ивентами: http://127.0.0.1:8000/events/
* Личный кабинет и рейтинг: http://127.0.0.1:8000/ranking/
* Админка: http://127.0.0.1:8000/admin/

## Коротко по функционалу:
* Во вкладке Мероприятия: Для участников - Возможность подать заявку на мероприятие. Для организаторов - Кнопка «Создать новое мероприятие» (заполнение даты, названия и баллов за участие)
* Во вкладке Рейтинг: Отображение участников с сортировкой по суммарным баллам или по конкретным направлениям (IT, Соц., Проекты, Медиа). Для инспектора: В таблице появляется дополнительная колонка «Паспорт»
* Во вкладке Личный кабинет: Отображение аватара, текущей роли и индивидуальных карточек с баллами по направлениям.
