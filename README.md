Инструкция по запуску:
1) Скачать файл
2) Открыть папку проекта и запустить терминал
3) В терминал написать команду python manage.py runserver
4) 
   а) Если необходим доступ к админке, то необходимо открыть в браузере  http://127.0.0.1:8000/admin/
   б) Если необходим доступ к пользовательской стороне сайта, то необходимо открыть  http://127.0.0.1:8000/events/
