# Hakoton Web Site

Система управления мероприятиями и рейтингом активистов. Проект написан на Django и включает в себя личный кабинет участника, общую таблицу лидеров и список актуальных событий.

## 🚀 Инструкция по запуску

### 1. Установка зависимостей
Убедитесь, что у вас установлен Python. Откройте терминал в папке проекта и выполните:
```bash
pip install -r requirements.txt
````

### 2\. Настройка базы данных

Если при запуске возникает ошибка `OperationalError: no such column`, это значит, что база данных на вашем компьютере не синхронизирована с кодом. Выполните команды миграции:

```bash
python manage.py makemigrations
python manage.py migrate
```

### 3\. Запуск сервера

Используйте соответствующую команду для вашей операционной системы:

**Для Windows (PowerShell/CMD):**

```powershell
python manage.py runserver
```

**Для macOS (Terminal):**

```bash
python3 manage.py runserver
```
