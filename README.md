# Инструкция по запуску

1. Скачать файл
2. Открыть папку проекта и запустить терминал
3. В терминале написать команду `python manage.py runserver`
4. Ссылки для доступа:
    * **Админка:** [http://127.0.0.1:8000/admin/](http://127.0.0.1:8000/admin/)
    * **Сайт:** [http://127.0.0.1:8000/events/](http://127.0.0.1:8000/events/)
